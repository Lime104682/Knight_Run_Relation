# 2. 게임 컨셉

플레이어는 특수 초상능력인 **인연모방**, **인연추출**, **인연동조**를 보유한 기사 교육생이다.

다양한 기사들과 인연을 맺으며 그들의 능력을 모방하고 성장하여 최종적으로 인류 최강 기사 반열인 영웅에 도달하는 것을 목표로 한다.

---

# 3. 세계관

인류는 괴수와 다양한 세력과의 전쟁을 이어가고 있다.

플레이어는 기사 교육생으로서 파견함선에 소속되어 여러 행성을 탐사하고 전투를 수행하며 성장한다.

스토리는 원작 나이트런의 주요 사건들을 따라가며 진행된다.

---

# 4. 플레이어 시스템

## 플레이어 특징

플레이어는 일반 캐릭터와 달리 전용 성장 시스템을 보유한다.

### 초상능력

### 인연모방

상대의 능력과 기술을 관측 및 분석하여 자신의 능력으로 복제하는 초상능력

획득 가능 요소

- 액티브 스킬
- 패시브 스킬
- 궁극 스킬
- 함선 스킬

---

### 인연동조

모방 대상이 파티 내에 존재할 경우 해당 대상과 플레이어의 능력을 동시에 강화한다.

예시

- 궁극기 피해량 증가
- 쿨타임 감소
- 추가 효과 부여

---

### 인연추출

인연이 깃든 대상에게서 인연석을 추출하는 능력

인연석 사용처

- 캐릭터 성장
- 궁극 스킬 강화
- 인연모방 강화
- 잠재력 해방

---

## 플레이어 장비

### AB 소드(플레이어용)

플레이어 전용 무장

획득 방법

- 제작
- 이벤트 보상
- 스토리 진행

강화 요소

- 공격력
- 특수효과
- 전용 스킬

# 7. 캐릭터 등급

| 등급 | 단계 |
| --- | --- |
| 1성 | 교육생 / 견습기사 |
| 2성 | 평기사 / 77형 |
| 3성 | 베테랑 / 5형 |
| 4성 | 마스터 나이트 / 보통의 5형 정도는 압살, 대 영식전의 주전력 해당 |
| 5성 | 기사단장 / 부단장 / 지상전 한정 고랭크 영식과 단기 접전을 벌일 수 있는 정도 |
| 6성 | 탑소드 / A랭크 영식에게 우위를 점하며 AAA급 이상의 레전드급 괴수와 비교됨 |

※ 파워밸런스는 해당 나무위키 참고 : 상위괴수와의 파밸#s-4)

※ 성급은 해당 인물이 잠재력을 완전히 개화했을 경우 도달 가능한 수준을 의미한다.

예) 프레이 마이어 - 6성 , 앤 마이어 - 6성 , 릭 맥코이 - 4성 , 질 맥켈런 - 4성 등…
